# D-08 部署运维说明

## 环境变量

部署前需要配置：

- `MYSQL_HOST`
- `MYSQL_PORT`
- `MYSQL_DATABASE`
- `MYSQL_USERNAME`
- `MYSQL_PASSWORD`
- `JWT_SECRET`
- `VITE_API_BASE_URL`

## 端口

- 后端：8080
- 前端开发服务器：5173
- 远程 MySQL：3306
- 可选 Redis：6379

## 启动验证

1. 启动后端并访问 `/actuator/health`。
2. 打开 Swagger UI。
3. 启动前端并访问登录页、门店页。
4. 检查 Flyway 是否完成迁移。

## 常见故障

- MySQL 连接失败：检查安全组、白名单、端口、用户名和库名。
- Flyway 失败：检查远程账号是否具备 DDL 权限。
- 前端接口失败：检查 `VITE_API_BASE_URL` 和后端跨域配置。
