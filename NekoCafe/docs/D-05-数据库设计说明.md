# D-05 数据库设计说明

## 数据库选型

项目使用远程 MySQL 8，存储引擎为 InnoDB，字符集为 utf8mb4。数据库迁移由 Flyway 管理。

## 核心表

第一阶段 MVP 表包括：

- 用户权限：`user`、`role`、`user_role`、`member_account`
- 门店桌位：`store`、`dining_table`、`reservation_slot`
- 预约订单：`reservation`、`food_order`、`food_order_item`、`payment_record`
- 菜品：`dish_category`、`dish`
- 看板：`dashboard_stat`

## 并发控制

预约创建通过 `reservation_slot` 的条件更新防止超约：只有当 `available_count > 0` 时才扣减余量，并通过 `version` 字段支持乐观锁设计。

## 安全

应用不使用 root 账号连接数据库。真实连接信息通过环境变量提供，备份文件和 `.env` 不提交到 Git。
